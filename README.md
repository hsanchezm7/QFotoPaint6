## Acerca del proyecto

**QFotoPaint** es una aplicación de escritorio para el procesamiento de imágenes y vídeo
desarrollada en C++, utilizando el framework Qt y la librería OpenCV.

Inspirada en herramientas reales de edición como GIMP o Adobe Photoshop, QFotoPaint permite al
usuario abrir, visualizar y editar múltiples imágenes simultáneamente, aplicando diferentes
filtros, transformaciones y efectos de procesamiento digital de imagen de forma sencilla e intuitiva.

## Requisitos

QFotoPaint requiere tener instalado un compilador compatible con el estándar C++17 o superior.

Es necesario disponer del entorno de desarrollo [Qt Creator](https://www.qt.io/download), junto con el framework Qt y la
librería [OpenCV (versión 4.10.0)](https://github.com/opencv/opencv/releases/tag/4.10.0), correctamente configurados para compilar y ejecutar la aplicación.
